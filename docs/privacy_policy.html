Kebijakan Privasi

Dompi tidak mengumpulkan data pribadi. Semua data transaksi disimpan secara lokal di perangkat Anda menggunakan basis data SQLite. Aplikasi tidak melakukan transmisi data ke server pihak ketiga.

Jika Anda menghapus aplikasi atau data aplikasi, seluruh data transaksi akan ikut terhapus dari perangkat.
